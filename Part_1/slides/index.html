Strings

A "String" is type of variable in Python that holds alphanumeric data.

Strings are created in Python by surrounding text with single or double quotes.


                        >>> 'spam and eggs'
                        'spam and eggs'
                        >>> 'doesn\'t'  # use \' to escape the single quote...
                        "doesn't"
                        >>> "doesn't"  # ...or use double quotes instead
                        "doesn't"
                        >>> '"Yes," he said.'
                        '"Yes," he said.'
                        >>> "\"Yes,\" he said."
                        '"Yes," he said.'

Strings

Strings can span more than one line.

The '\n' character means "new line".


                        >>> two_line_string = 'This is line 1.\nThis is line 2.'
                        >>> two_line_string
                        'This is line 1.\nThis is line 2.'
                        >>> print(two_line_string)
                        This is line 1.
                        This is line 2.

Strings

Another way to create multi-line strings is with triple-quotes.

Triple-quotes let you leave out the '\n' character.


                        >>> usage = """\
                        ... Usage: thingy [OPTIONS]
                        ...      -h                        Display this usage message
                        ...      -H hostname               Hostname to connect to
                        ... """
                        >>> print(usage)
                        Usage: thingy [OPTIONS]
                            -h                        Display this usage message
                            -H hostname               Hostname to connect to

Strings

Strings can be glued together with the + operator.


                        >>> prefix = 'super'
                        >>> suffix = 'duper'
                        >>> prefix + suffix
                        'superduper'
                        >>> one = '1'
                        >>> two = '2'
                        >>> one + two
                        '12'

String Indexing

We can use square brackets ([]) to access an individual character from a string.


                        >>> word = 'Python'
                        >>> word[0]  # character in position 0
                        'P'
                        >>> word[5]  # character in position 5
                        'n'

This is called indexing. The index of the first element is always 0.

String Indexing

Indexes can be negative! In this case they start from the end of the string.


                        >>> word = 'Python'
                        >>> word[-1]  # last character
                        'n'
                        >>> word[-2]  # second-last character
                        'o'
                        >>> word[-6]
                        'P'

String Slicing

The [] notation can also be used to get a substring, or slice of a string.

[i:j] where characters starting at position i are included, up to position j (which is excluded).


                        >>> word = 'Python'
                        >>> word[0:2]  # 0 (included) to 2 (excluded)
                        'Py'
                        >>> word[2:5]  # 2 (included) to 5 (excluded)
                        'tho'

String Slicing

i, j, or both may be omitted when creating a slice.


                        >>> word = 'Python'
                        >>> word[:2]
                        'Py'
                        >>> word[2:]
                        'thon'
                        >>> word[:]
                        'Python'
                        >>> word[:2] + word[2:]
                        'Python'
